# CRUD em JAVA

Crud basico em java via console, com o intuito de exercitar as habilidade desenvolvimento, realizando integração com Bando de Dados MySQL, ultilizando JDBC e estruturação de classes, DAO, Factory

## CRUD
  - Create
  - Read
  - Update
  - Delete
